# Jaya Karya Konstruksi Company Profile Website

A company profile website developed to support the digital presence of Jaya Karya Konstruksi.  
This project focuses on presenting company information, services, project documentation, and public-facing content in a structured web platform.

## Tech Stack

- Laravel
- React.js
- Inertia.js
- Tailwind CSS
- Vite
- TypeScript
- MySQL

## Main Features

- Company profile pages
- Service information pages
- Project/gallery content
- Responsive layout
- Content management support
- Image handling
- SEO sitemap support

## Project Status

This repository is used as a portfolio showcase.  
Some implementation details, environment configuration, deployment setup, and internal business logic are intentionally omitted for security and privacy reasons.

## Repository

This project was developed as part of a real-world company website implementation and portfolio documentation.
